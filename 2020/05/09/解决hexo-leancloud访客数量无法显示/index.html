解决hexo leancloud访客数量无法显示

Posted on 2020-05-09 | Post modified: 2020-05-09 | In note | | Visitors:
Words count in article: 144 | Reading time ≈ 1

在我添加域名之后，访客数量不显示了，还在网站下方显示

Code 403: 访问被API域名白名单拒绝,请检查你的安全域名设置.

原因：是leancloud安全访问设置的网站出问题了

解决办法：修改web安全域名，添加个人域名和GitHub域名，格式跟这个保持一致

还是会存在小问题，如果直接输入 自己的域名访问，不添加http:// 也是不会显示访客的
